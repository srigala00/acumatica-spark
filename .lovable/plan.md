

# Ganti Logo Header & Footer dengan File PNG

Ya, bisa! Anda cukup upload file PNG logo melalui chat ini, lalu saya akan:

1. Simpan file logo ke folder `public/` (misalnya `public/logo.png`)
2. Update `src/components/layout/Header.tsx` — ganti blok div logo (⚡ + teks "PLN Suku Cadang") dengan `<img src="/logo.png" />`
3. Update `src/components/layout/Footer.tsx` — sama, ganti blok logo dengan `<img>`
4. Update `src/components/layout/AdminSidebar.tsx` — ganti logo di sidebar admin

**Yang perlu Anda lakukan:** Upload file PNG logo Anda di chat ini, lalu saya akan langsung pasang.
